# 🚀 TUK LIFE OS v4 Professional

Mobile-first Personal Operating System for TUK.

## Run in GitHub Codespaces

```bash
npm install
npm run dev
```

Open **PORT 3000**.

## Core Modules
- Home Dashboard
- AI Brain
- Smart Calendar
- Finance OS
- Health & Gym
- Satys Work
- Maintenance System
- Vehicle Manager
- Travel Planner
- Knowledge Base
- Analytics
- Settings

## Architecture
React + TypeScript + Vite + Pure CSS. No Tailwind native dependency, so it avoids the `@tailwindcss/oxide` Codespaces error.
