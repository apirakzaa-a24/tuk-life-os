# TUK OS v5.1 Connected Fitness + Travel ERP

Local-first working package สำหรับใช้งานจริงในเครื่องก่อนต่อ Cloud Sync ภายหลัง

## สิ่งที่ทำได้แล้ว

- Dashboard รวมข้อมูลจากฐานข้อมูลกลางตัวเดียว
- เพิ่ม / แก้ไข / ลบ / ค้นหา ข้อมูลได้จริง
- บันทึกลง LocalStorage และ refresh แล้วข้อมูลยังอยู่
- Export / Import Backup JSON
- Reset Seed Data
- เลือกโมดูลหลักและหัวข้อย่อยได้ละเอียด
- เพิ่มระบบ `linkedTo` ให้ข้อมูลข้ามโมดูลแสดงร่วมกันได้ เช่น Travel expense เชื่อม Finance, Fitness workout เชื่อม Health/Goals
- เพิ่ม Travel Command Center แบบละเอียดมาก
- เพิ่ม Fitness ERP แบบละเอียดมาก
