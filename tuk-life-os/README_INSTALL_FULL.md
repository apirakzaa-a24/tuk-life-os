# TUK LIFE OS v1.0 Professional Full Mobile Ready

## วิธีติดตั้ง
1. แตก ZIP
2. นำไฟล์ทั้งหมดไปทับโปรเจกต์เดิมที่ `Documents/GitHub/tuk-life-os/tuk-life-os/`
3. เปิด Codespaces หรือ Terminal
4. รัน:

```bash
npm install
npm run dev
```
