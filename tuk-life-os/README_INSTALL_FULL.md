# TUK LIFE OS v7 Professional Full Release

## Install
1. Backup old project folder.
2. Copy every file/folder from this release into `Documents/GitHub/tuk-life-os/tuk-life-os/`.
3. Replace all files when asked.
4. Run:
```bash
npm install
npm run dev
```

- TUK LIFE OS Professional
- Version 7.0.0
- Full Release v7
